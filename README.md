# vuejs_app

## Project setup
```
git clone the project
cd VUEJS_APP
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
App will be running on http://localhost:8080/
```

### Compiles and minifies for production
```
npm run build
```

### Run the database for testing
```
nodemon server
```

### Lints and fixes files
```
npm run lint
```
